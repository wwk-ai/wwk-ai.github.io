> **Java后端技术体系** — 深耕Java生态十余年，分享Spring全家桶、高并发架构、云原生实战与源码解析。

## 专栏导读

本专栏聚焦企业级Java后端开发的核心技术栈，从基础框架到高并发架构，从源码解析到线上排查，系统性分享实战经验：

- **Spring生态深度解析**：Spring Boot自动配置原理、Spring Cloud微服务治理
- **高并发系统设计**：限流熔断、分布式锁、缓存策略、消息队列选型
- **JVM与性能优化**：内存模型、GC调优、线上OOM排查、Arthas实战
- **云原生实践**：Docker容器化、K8s编排、服务网格、GitOps交付
- **中间件原理**：Redis、RocketMQ、ElasticSearch、ShardingSphere源码级分析
